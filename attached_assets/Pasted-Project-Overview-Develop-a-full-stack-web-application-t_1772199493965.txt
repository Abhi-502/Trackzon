Project Overview

Develop a full-stack web application that:

Accepts single or multiple product URLs

Scrapes product name and price from e-commerce pages

Stores price history in database

Compares current price with previously stored price

Sends email notification when price drops

Allows tracking for up to 1 month per user request

Supports CSV upload for bulk URLs

Supports multi-platform (Amazon, Flipkart, etc.)

🧠 Backend Processing Flow

User submits URL(s)

Backend validates URL

Detects platform (Amazon/Flipkart)

Calls appropriate scraper module

Extracts:

Product Name

Current Price

Product Image (optional)

Saves data to database

🔄 Price Comparison Logic

Every scheduled run:

Fetch latest price

Compare with last_checked_price

If new_price < last_checked_price:

Trigger email notification

Update last_checked_price

Insert into PriceHistory

📅 Scheduler

Use cron job / APScheduler

Run every:

6 hours (recommended)

Only check products where:

Current date < tracking_end_date
